# rtv1

## Project Overview

This project is about creating a simple raytracer from scratch.

## Installation

``` bash
$> git clone https://github.com/bmoiroud/rtv1.git && cd rtv1 && make
```

## Usage

``` bash
$> ./rtv1 <scene>
```

| Key     | Action             |
|:-------:|:-------------------|
| W       | move forward       |
| A       | move left          |
| S       | move backward      |
| D       | move right         |
| Q       | turn left          |
| E       | turn right         |
| Space   | move up            |
| CTRL    | move down          |
| esc     | quit               |
